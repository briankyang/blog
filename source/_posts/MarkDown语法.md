### 标题

`# 一级标题`
`## 二级标题`
.
`###### 六级标题`

---

### 列表

无序列表 : **行首为 `- `或`* `**

* C++
* JAVA


<!--more-->

有序列表 : 行首为**数字、点** 如 `1. `

1. C++
2. JAVA

***


### 引用

`>文字`

>文字

***

### 图片与链接

- 插入链接

`[名称](链接)`

- 插入图片

`![名称](链接)`

***

### 代码框

- 单行

`` `code` ``

- 多行

````
```lang
 code
```
````

***

### 表格

```
| Tables        | Are           | Cool  |
| ------------- |:-------------:| -----:|
| col 3 is      | right-aligned | $1600 |
| col 2 is      | centered      |   $12 |
| zebra stripes | are neat      |    $1 |
```
